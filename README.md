# Quantum Lab v2

[Italiano](#italiano) · [English](#english)

## Italiano

**Quantum Lab** è una Progressive Web App gratuita, bilingue e open source per generare password da misure quantistiche e analizzare esperimenti Bell–CHSH, GHZ, Mermin e QRNG.

### Caratteristiche

- interfaccia unica in italiano e inglese;
- generazione guidata di password da misure grezze QRNG;
- generatore crittografico locale del dispositivo;
- analisi Bell–CHSH, GHZ, Mermin e QRNG;
- importazione TXT, CSV e JSON;
- funzionamento offline dopo il primo caricamento;
- nessun account interno, tracciamento o caricamento dei dati;
- nessun backend e nessun costo obbligatorio.

### Quantum Inspire

La PWA apre Quantum Inspire e fornisce un circuito cQASM QRNG pronto da copiare o scaricare. L'esecuzione avviene con l'account personale dell'utente. Le credenziali non vengono mai inserite nella PWA.

### Pubblicazione

Il progetto è statico. Può essere pubblicato gratuitamente tramite GitHub Pages o Cloudflare Pages caricando tutti i file nella stessa cartella.

## English

**Quantum Lab** is a free, bilingual, open-source Progressive Web App for generating passwords from quantum measurements and analyzing Bell–CHSH, GHZ, Mermin and QRNG experiments.

### Features

- single Italian and English interface;
- guided password generation from raw QRNG measurements;
- local cryptographic device generator;
- Bell–CHSH, GHZ, Mermin and QRNG analysis;
- TXT, CSV and JSON import;
- offline support after the first load;
- no internal accounts, tracking or data uploads;
- no backend and no mandatory cost.

### Quantum Inspire

The PWA opens Quantum Inspire and provides a ready-to-use QRNG cQASM circuit. Execution uses the user's own account. Credentials are never entered into the PWA.

### Deployment

This is a static project. It can be hosted free of charge with GitHub Pages or Cloudflare Pages by uploading all files to the same directory.
